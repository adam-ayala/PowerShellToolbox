# Set-PSExecutionPolicy

## SYNOPSIS
Sets the PowerShell Execution Policy and Scope for the current session

## SYNTAX

```
Set-PSExecutionPolicy [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
Sets the PowerShell Execution Policy and Scope for the current session

## EXAMPLES

### EXAMPLE 1
```
Set-PSExecutionPolicy
```
