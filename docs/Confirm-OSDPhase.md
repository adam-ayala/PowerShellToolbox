# Confirm-OSDPhase

## SYNOPSIS
Verifies the Windows Operating System deployment phase.

## SYNTAX

```
Confirm-OSDPhase [<CommonParameters>]
```

## DESCRIPTION
Verifies the Windows Operating System deployment phase.
WinPE, OOBE, Specialize, Audit Mode or Windows

## EXAMPLES

### EXAMPLE 1
```
Confirm-OSDPhase
```

Confirms the Operating System Deployment phase
