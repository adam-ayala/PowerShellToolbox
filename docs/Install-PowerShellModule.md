# Install-PowerShellModule

## SYNOPSIS
Installs a PowerShell Module

## SYNTAX

```
Install-PowerShellModule [-Name] <String> [<CommonParameters>]
```

## DESCRIPTION
Installs a PowerShell Module

## EXAMPLES

### EXAMPLE 1
```
Install-PowerShellModule -Name MyModule -Force
```

Force installs the MyModule PowerShell Module

## PARAMETERS
